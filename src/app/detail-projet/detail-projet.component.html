Voici le travail Angular réalisé par: Mohamed Aiman Benomar et Lahjouji Ayman

Ce travail comporte les éléments demandés suivants:

- Authentification à l'aide de Json Web Tokens (JWT) en nous basant sur le tuto donné plus d'autre

- Ajout de nouvelles propriétés au modèle des Assignments

- Affichage dans un tableau Angular Material avec un header fixe

- La possibilité d'ajouter un assignement dans les favoris

- Les formulaires d'ajout et de détails proposeront un choix fixe de matières

- Ajout d'un component pour naviguer en haut de la page étant donné que nous n'avons pas réussi à faire la pagination

- Des pop-ups pour informer l'utilisateur de la connexion et la deconexion (toast)

- Hébergement sur Heroku.com

Ce projet est basé sur le code que vous nous avez fourni et les élément ajoutés sont presque tous bases sur les informations de https://material.angular.io/ plus Tuto youtube pour les favoris et Tuto Authentification JWC node
